Practical penetration testing documentation from hands-on labs. This repository showcases structured methodology, technical clarity, and defensive awareness suitable for **cybersecurity internships**, **SOC analyst** applications, and **junior penetration testing** roles.

---

## About

Each write-up follows a consistent, recruiter-friendly format:

- **Reconnaissance** — host and service discovery  
- **Enumeration** — protocol- and application-specific mapping  
- **Exploitation** — initial access and proof of compromise  
- **Privilege escalation** — local privesc to root  
- **Key takeaways** — lessons learned and remediation  

Screenshots, command snippets, and summary tables are included for quick review by hiring managers and technical interviewers.
